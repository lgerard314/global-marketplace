# Global Claude Marketplace

Private plugin marketplace for logan's Claude Code plugins.

## Included plugins
- global-plugin
- board — anti-staleness governance for Claude-Code-driven planning under `BOARD/plans/` ([repo](https://github.com/lgerard314/board-plugin))

## Add this marketplace
In Claude Code:

/plugin marketplace add https://github.com/lgerard314/global-marketplace

## Install a plugin
/plugin install global-plugin@global-plugins
/plugin install board@global-plugins

## Update marketplace
/plugin marketplace update
